import GraphiQL from '../../components/GraphiQL';

Plugins are a core tenant in every WordPress site. WPGraphQL provides native queries and mutations
for interacting with Plugins via GraphQL.

### Queries

<GraphiQL query="{
  plugins {
    nodes {
      id
      authorUri
      pluginUri
      description
      version
      author
    }
  }
}"
/>



<GraphiQL query="{
  plugin(id: 1) {
    id
    authorUri
    pluginUri
    description
    version
    author
  }
}"
/>
